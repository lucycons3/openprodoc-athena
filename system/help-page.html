Help Page

Athena comes with a template to build a help page for your users.

This template provides an index with some built-in functionalities, like CSS-based smooth scrolling.

Create custom pages

This template can also be used to add custom pages to your OpenProdoc site, like a page about the history of the institution, an acknowledgement page, etc.

Then you can link the new pages in the navigation bar.

Lists

This template supports ordered lists:

  1. Owl.
  2. Hummingbird.
  3. Jay.

You can create unordered lists too:

  • Owl.
  • Hummingbird.
  • Jay.

Galleries

Thanks to its integration with baguetteBox.js, Athena helps you to easily add image galleries to your OpenProdoc site.
